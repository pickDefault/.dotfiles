        -- D to [D]uplicate

    -- Ctrl + W to move and focus between [W]indows
        -- H, J, K, L to move between split editors
        -- Z to focus editor group
        -- 0-9 to select window
        -- 
    -- Ctrl + Shift + = (Which is actually Ctrl + '+') to increase size
    -- Ctrl + - to decrease size
    -- Command + K prefix
        -- M to select language
    -- Space to enter code navigation and manipulation
